🧪 技术演示

像播视频一样嵌入 GalGame

这个页面展示了 mountGamePlayer(container, storyData) 的能力。
点击下方按钮，一个完整的 GalGame 实例将嵌入到这个页面中。

无需 iframe、无框架锁定、每个实例完全隔离。

🎯

任意容器挂载

传入任意 DOM 元素，游戏就在哪里渲染。不依赖特定 #app 元素。

📦

数据即参数

游戏数据通过 props 注入，不硬编码 import。同一个页面内可挂载不同数据。

🧹

完整生命周期

unmount() 可完全销毁实例，不残留事件监听或 DOM。

import { mountGamePlayer } from '@galgame/game';

const player = mountGamePlayer('#container', storyData);

player.getEngine() // → GalEngine 实例，可直接控制
player.unmount() // → 完全销毁
